## ⚙️ How This Repository Works

This repo is powered by **NeetCode GitHub Sync**, which automates the workflow:

- 🔄 **Auto Sync** — Every submission on NeetCode is automatically pushed here  
- 📦 **Bulk Import** — Past solutions can be synced in one go  
- 🛠 **Manual Control** — Individual submissions can be added/removed  
- 🎯 **Filtering** — Option to sync only accepted solutions
